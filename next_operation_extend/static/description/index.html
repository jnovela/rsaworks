Next Operation Customization

Next Operation Customization - module will prevent the work order of being procced if the setting for that is true and preceding work orders are not finished.

In below image you can find routing and operations related to that routing.

We have to enable a boolean "Is All Preceding WO Complete" as "True" in the operation for which you want all preceding operations to get complete before processing current operation.

For e.g right now we have set the boolean true for the "Long time assembly", so this operation's work order will get processed only when all the preceding operation's work orders will finished.

In below image you can see the created work orders for one manufacturing orders.

If we will try to process work order for "Long time assembly",and if the preceding work orders are not finished,then it will raise this warning.
